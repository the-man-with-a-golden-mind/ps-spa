# basic

This is a standalone example app for `ps-spa`.

Run it from this directory:

```bash
bun install
bun run dev
```

or:

```bash
npm install
npm run dev
```

Build a production bundle from this directory:

```bash
bun run build
```

Routes in this example:

- `/`
- `/not-found`
- `/playground`
- `/people/:name`
