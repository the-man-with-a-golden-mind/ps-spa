# tailwind

This is a standalone example app for `ps-spa` with the `tailwind` page template enabled.

Run it from this directory:

```bash
bun install
bun run dev
```

or:

```bash
npm install
npm run dev
```

Build a production bundle from this directory:

```bash
bun run build
```

Routes in this example:

- `/`
- `/not-found`
- `/marketing/hero`
- `/guides/:slug`
